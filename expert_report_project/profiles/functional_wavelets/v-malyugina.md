## METODOLOGÍAS WAVELETS Y FUNCIONAL DATA

### Algoritmos preferidos
- Maximal Overlap Discrete Wavelet Transform (MODWT) - frecuencia: alta
- Wavelet variance analysis - frecuencia: media

### Críticas metodológicas
- Critica: Modelos ARIMA que no capturan heterocedasticidad multiescala
- Rechaza: Análisis de frecuencia pura sin resolución temporal
- Advertencia sobre: Estacionalidad en datos económicos

### Software desarrollado/recomendado
- [R waveslim, Python PyWavelets]
- Lenguajes: R, Python
- Herramientas de visualización: Wavelet variance plots
