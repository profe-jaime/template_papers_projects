## METODOLOGÍAS WAVELETS Y FUNCIONAL DATA

### Algoritmos preferidos
- Continuous Wavelet Transform (CWT) - frecuencia: alta
- Wavelet-based time-frequency analysis - frecuencia: alta

### Críticas metodológicas
- Critica: Análisis de Fourier que pierden resolución temporal
- Rechaza: Métodos que no consideran evolución espectral
- Advertencia sobre: Artefactos de cone of influence

### Software desarrollado/recomendado
- [MATLAB Time-Frequency Toolbox, Python tftb]
- Lenguajes: MATLAB, Python
- Herramientas de visualización: Scalograms y time-frequency plots
