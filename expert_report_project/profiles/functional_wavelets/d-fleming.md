## METODOLOGÍAS WAVELETS Y FUNCIONAL DATA

### Algoritmos preferidos
- DWT (Discrete Wavelet Transform) - frecuencia: alta
- Análisis de escalamiento - frecuencia: media

### Críticas metodológicas
- Critica: Ignorar no estacionariedad en datos financieros porque distorsiona la detección de patrones
- Rechaza: Métodos tradicionales sin multiresolución cuando hay cambios de escala
- Advertencia sobre: Sobreinterpretación de coeficientes sin contexto económico

### Software desarrollado/recomendado
- [MATLAB Financial Toolbox, Python PyWavelets]
- Lenguajes: MATLAB, Python
- Herramientas de visualización: Gráficos de coeficientes wavelet vs tiempo
